## Project Overview
This project is an interactive narrative website inspired by The Weeknd’s 10 album eras. Each album page uses its own mood, color palette, visuals, layout, and interaction to create a unique immersive experience.

## Technologies Used
- HTML
- CSS
- JavaScript
- Canvas creative coding
- Responsive design with media queries
